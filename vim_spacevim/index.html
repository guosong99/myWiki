〇 写在前面的话

1.目标不是要成为spacevim专家，所以不必在意细节 2.目标是熟练操作，提高工作效率，顺便成为spacevim高手

一 高频操作汇总

#配置与更新
:SPUpdate   更新所有插件，包括spacevim。
space-f-v-d 打开用户配置文件，怎么返回或单独关闭配置
space-f-v-v 查看spacevim版本
:h SpaceVim 访问SpaceVim帮助文档

\   leader key
b   buffer快捷键前缀
p   project快捷键前缀
s   search快捷键前缀
h   help快捷键前缀

#buffer操作： 
SPC <Tab> //切换前一buffer,或来回切换
SPC b . 启用缓冲区临时快捷键
SPC b n //跳转到下一个buf
SPC b p //跳转到前一个buf
SPC b h //打开spacevim home界面
SPC b d //删除当前缓冲区，但保留编辑窗口
SPC b D  //选择一个窗口，并删除其缓冲区
SPC b c 删除其它已保存的缓冲区
SPC b P 使用系统剪切板内容替换当前缓冲区

#文件操作
SPC f t 打开当前文件所在目录树=F3
SPC f o 代开文件树，并定位到当前文件
SPC f f 打开文件
SPC f F 打开光标下的文件
SPC f / 使用 find 命令查找文件，支持参数提示
SPC f b 跳至文件书签，如何设置书签
SPC f d Windows下显示/隐藏磁盘管理器
SPC f s 保存文件
SPC f S 保存所有文件
SPC f r 打开文件历史
SPC f y 复制并显示当前文件的绝对路径

文件操作

一下操作都在 vimfiler 窗口进行 （F3 打开）

N：新建文件 .：切换显示隐藏文件 yY：复制文件 yy：复制文件路径 P：粘贴文件 i：显示文件夹历史 sg：垂直分屏打开文件 Ctrl+r：刷新文件树

其他操作

\y 复制, \p 粘贴 （比 "+y, "+p 方便）

项目技巧 C/C++项目 在你项目的根目录新建一个 .clang 文件，在文件里加上 clang 的参数，如-I.. -std=c++11 等。这样你打开项目里的任何文件，根目录都是 .clang 文件所在的目录，而且语法检查也依赖这个文件。 类C代码增强高亮在 gbk 文件中可能会出现问题，这时使用 :ChromaticaStop 停掉增强高亮即可。

工程管理project

首先是工程（根）目录要有.git/目录,命令以SPC p 前缀

SPC p p #显示所有工程
SPC p f #在当前工程中查找文件
SPC p / #在当前工程中搜索文件（fuzzy search for text)
SPC p k #关闭当前工程的所有缓冲区
SPC p t #find project root

代码浏览

F2：打开语法树，F3：打开文件树 SPC '：打开 shell SPC m g c：创建 gtags database SPC m g d：查找定义 SPC m g r：查找用 SPC m g u：更新 gtags database Ctrl ]：查找定义（有 tags 文件时） Ctrl o：返回上一次位置 SPC e l：打开代码检查错误窗口 SPC e L：打开代码检查错误窗口并聚焦 SpaceVim 的 root 目录不在当前文件目录时（如git,svn项目），此时按 F2 打开语法树会没有内容，因为 F2 只会在一层目录生成 tags 文件，此时需要到命令行在你需要的位置执行 ctags * -R

2.ubuntu

四 spacevim配置

1.字体配置

#Linux/Unix: 
:set guifont=Monospace\空格14 #注意这里需要对空格使用\进行转意 
#Windows: 
:set guifont=Monospace:h14 #注意这里的字体大小需要有h的标识

可以配置自己喜欢的字体和字体大小，默认spacevim使用 SauceCodePro Nerd Font Mono字体，在配置文件 ~/.SpaceVim.d/init.toml设置

[options]
    guifont = "SauceCodePro Nerd Font Mono:h11"

2.更新spacevim

（1）在.SpaceVim目录下,执行git pull手动更新 （2）git -C ~/.SpaceVim pull // 直接更新 （3）不建议自动更新，影响启动速度。如有必要，可以在配置文件里添加automatic_update = true
（4）如通过插件管理器来更新，在vim界面执行:SPUpdate SpaceVim （5）:SPUpdate 更新所有插件，包括spacevim。可跟参数更新具体插件（TAB可补全）可同时添加多个插件名称作为参数
